# 자료구조와 알고리즘을 공부해야 하는 이유

무엇을 만들고 배포도 끝났으며 소스코드에 버그도 없다.

그러나 너무 느리다.

<br>

자료구조와 알고리즘은 이러한 문제를 개선해 줄 수 있다.

<br>

**알고리즘**은 여러개의 지시사항이다.

어떤 액션을 수행하기 위해 컴퓨터가 수행해야하는 것들이다.

**자료구조**는 데이터를 정리하는 방법이다.

어떻게 정리하느냐에 따라 스피드에 큰 영향을 준다.

<br>

<br>

<br>
